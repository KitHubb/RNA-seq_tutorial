## 초심자의 흔한 실수 들 
1. TPM/FPKM를 DESeq2 input으로 사용  
  → DESeq2/edgeR는 raw count 기반 NB model 가정  

2. PCA에서만 batch correction 수행  
  → visualization correction ≠ DEG correction  
  → design formula에 batch 포함 필요  
  → Leek et al., 2010; Johnson et al., 2007

3. low replicate (n=2~3) 결과 과신  
  → dispersion estimation 불안정, reproducibility 저하  

4. technical replicate를 biological replicate처럼 사용  
  → pseudo-replication 발생  

5. normalization 개념 없이 CPM/TPM/TMM 사용  
  → RNA-seq는 compositional 특성 존재  


6. volcano plot만 보고 biological conclusion 도출  
  → pathway/context/baseMean 고려 부족  

7. DEG만 보고 effect size(log2FC) 무시  
  → statistical significance ≠ biological significance  

8. confounded design 무시  
  → batch와 condition 완전 overlap 시 해석 불가능
